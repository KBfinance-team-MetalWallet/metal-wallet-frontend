# 👜 프로젝트 소개

### 암표방지 + 전자지갑 = 💪🏼강철지갑

'강철지갑'은 사용자가 QR로 결제하고 계좌를 확인할 뿐만 아니라, 티켓을 예매하고 안전하게 사용할 수 있는 플랫폼을 제공하는 서비스입니다.

<div align="left">
